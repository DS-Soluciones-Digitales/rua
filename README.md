# Currency Converter - DS Soluciones Digitales

Este proyecto es una solución técnica para el proceso de selección de Desarrolladores Marzo - Abril 2026. Consiste en un conversor de divisas dinámico construido con **Angular 20**.

## 🚀 Características
- **Conversión en tiempo real**: Cálculo bidireccional entre USD, EUR, PEN y CNY.
- **Validación de Reglas de Negocio**: Bloqueo estricto de conversiones USD <-> CNY.
- **Gráfica de Tendencia**: Visualización de los últimos 5 tipos de cambio mediante CSS nativo.
- **Mocking**: Emulación de endpoint mediante un servicio inyectable.
